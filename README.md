# ADMET_Challenge_2025
This repository contains scripts for training models for the OpenADMET + ExpansionRx Blind Challenge.

## data
This directory contains the training data for the challenge. Separate subdirectories contain the 10 fold cross validation splits used to evaluate models.

## chemprop_results
This directory contains the scripts for evaluating differenct architectures based on the chemprop framework. The scripts are here to provide support for how the models were developed.
